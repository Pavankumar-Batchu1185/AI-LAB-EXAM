Steps to Run PROLOG Program in SWI-Prolog

1) Install SWI-Prolog
Download and install SWI-Prolog from official site:

2) Open Terminal / Git Bash
type: swipl

You will see:
Welcome to SWI-Prolog

3) Load the Prolog File
If your file is familytree.pl:
['familytree.pl'].

⚠️ Important:

file must be in current folder
include quotes ' '
end with .

4) Execute Query
Example:
father(john, paul).
Output:
true.

5) Exit SWI-Prolog
halt.
